                          INFORMACIÓN TÉCNICA
================================================================================

Base de Datos:     Neon Postgres

ORM:               Drizzle ORM
Estado:            ✓ Producción

================================================================================
                           NOTAS IMPORTANTES
================================================================================

• Las contraseñas están hasheadas de forma segura en la base de datos
• Cada usuario tiene un rol asignado (admin, docente, estudiante)
• Los docentes y estudiantes están asociados a "Ingeniería Agronómica"
• El sistema soporta múltiples carreras y períodos académicos
• Las sesiones persisten mediante cookies seguras
